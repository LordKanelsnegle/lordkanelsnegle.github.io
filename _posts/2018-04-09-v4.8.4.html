New Features:

  • Added mood command which will output Cinnamon's daily mood
  • deletemsg configuration option (toggle, disabled by default). When enabled, the message sent to invoke a command will be deleted when the bot carries out the command

  • Fortnite API

    Quite simply, Cinnamon can now retrieve a user's fortnite stats.
    For more information see the commands page.

Bug Fixes and Adjustments:

  • Removed modrole shenanigans in place of just using Discord's Admin perms. In other words, the modrole command has been removed and moderation commands can now only be used by server admins
  • All game commands which are restricted to the game's creator can now also be used by admins
  • Squished a bunch of bugs for the Fortnite API and CAH game mode, as well as improving efficiency
  • Added stats as a command alias for info
  • Fixed a bug that broke the shipping algorithm
  • Updated config command and made it look nicer
  • fortnite profile command now automatically finds the right platform instead of taking it as a parameter
  • Fixed waifurate values whcih were set higher than 10 (e.g. 35/10)
  • Fixed RNG lock (guarantees randomly picked gifs aren't the same as the last randomly picked thing)
  • Farewell messages should now behave as they should instead of sending welcome messages
  • Sprinkled with love and affection

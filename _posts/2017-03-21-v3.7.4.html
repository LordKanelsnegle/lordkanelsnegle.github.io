New Features:

  • Log channel will be set to the first text channel found with the word "log" in its name by default
  • Music channel will be set to the first voice channel found with the words "music" in its name by default
  • You can now use user-based commands on yourself without supplying your own username (e.g. avatar instead of avatar @YourUsername)
  • quote command now also quotes images (png, jpg, gif, tif, bmp, or ico)
  • Two separate cooldowns for fighting trigger words (e.g. fite me) and lewd words
  • Added new reactions to the aforementioned trigger words
  • Added more lewd words to the list of words that can trigger a lenny face
  • Made a new command, user info - use help to learn more or try it out yourself
  • Created a birthday command for setting your birthday (only the day and month, year is optional) which shows up in the above command
  • Created a bio command for setting a description about yourself, shown in your user info
  • Bot will now wish you a happy birthday if you've set a birthday
  • Made a dicklength command - 3 guesses as to what it does xd
  • Now possible to create and remove custom user joined/left messages using joinmsg and leavemsg. Set the channel for these messages using msgchannel

Bug Fixes and Adjustments:

  • Unfortunately, the rainbow feature had to be removed because in the new ToS it's considered an "abuse of the Discord API" :(
  • MMR from battling will now only be gained if the users you beat aren't AFK
  • Fixed a bug where gaydar could be set to negative values
  • Several micro-optimisations
  • Made the config command look cleaner and easier to read
  • Fixed a bug where the the bot would accept the same channel multiple times as an input for channel-based commands
  • Number of command restricted channels per server is now limited to 3
  • Removed the test command because it was useless
  • Broadcasts are no longer sent to guilds where the bot is disabled
  • Bot disabling/enabling now works properly
  • Music command channels can now be separate from regular command channels
  • Changed the way the bot handles disconnects so that it's less problematic for users
  • Fixed a bug where the bot wouldn't leave a vc if there was no one else with it (it should've)
  • Removed the user rank command
  • msgchannel and logchannel commands can now be set to "off" to disable either of those features
  • Sprinkled with love and affection

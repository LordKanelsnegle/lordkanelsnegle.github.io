New Features:

  • autorole delete will now also remove the autorole from every user that has it if the optional parameter fully (see commands page for full details)
  • Attempting to use a command in a non-command channel will now delete the command message, send an error, and delete the error as well after a 5 second delay

Bug Fixes and Adjustments:

  • Temporarily disabled the battle command
  • Fixed a bug causing autoroles created with message conditions to fail
  • Fixed a bug that caused the bot to hang on boot because of the way it used to handle autoroles
  • As a result of the above fix I also found the same problem with cmdchannels which I'd just gotten lucky with
  • Vastly improved the performance of autorole add should now execute much faster
  • Fixed a bug that limited timespans to 24h (affects VC stats and autoroles)
  • Resolved an issue in which autoroles were breaking if added without the Manage Roles perm and if user positions were higher than the bot's position
  • VCs will now be considered started when there are at least 2 non-bot users and finished when there is only one non-bot user left in the VC (because a bunch of loners were just AFKing in channels for ages which isn't really a "VC")
  • Timespans can now be in terms of weeks as well, with a new limit of 68 years, and is computed exceptionally faster while being even more forgiving of typos/input format
  • Users who have suggested features will no longer be mentioned in credits but will instead be granted the "Contributor" role on the support server (because it was getting too difficult to keep track of all of the users, especially since they change their names often)
  • Solved a problem that made the bot handle events multiple times (depending on how many times it has reconnected to Discord)
  • Fixed a typo for cmdchannel errors that said "There are no welcome messages [...]" instead of "There are no command channels [...]"
  • Fixed a really annoying ship bug that caused the command to break if usernames were too short, I've known about it for a while but only managed to figure it out this patch
  • Clarified on the main website that parameters containing spaces need to be wrapped with quotes if other arguments are given, and that roles/channels can be used directly as parameters instead of writing their names (@role or #channel instead of the name)
  • Resolved an issue that caused connect4 games to time out even if the game had been ended prematurely
  • Sprinkled with love and affection

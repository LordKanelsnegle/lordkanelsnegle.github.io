New Features:

  • New command insult allows you to spit fire at a user
  • waifurate command has been completed
  • ship command has been completed
  • New remindme command allows you to set reminders up to a year from the present

Bug Fixes and Adjustments:

  • Optimised portions of code regarding how the bot handled new guilds
  • Improved paginator introduced in the previous update
  • Bot will no longer delete guild configurations after leaving the guild
  • Fixed a bug that broke the bot's database
  • The info command now shows only actual users (no bots and users are only counted once even if they share multiple guilds with the bot)
  • responses is now a toggled command, not "on" or "off"
  • The glitter keyword now only triggers if said alone
  • If the bot takes more than a minute to reconnect after going down it'll now automatically reboot
  • The bot will now disable itself when the test bot is present and online in the same guild
  • Improved the battle command and added basic cheat prevention
  • Edited the help command to be more informative
  • n00dles command now has aliases n00dle and noodle
  • Sprinkled with love and affection

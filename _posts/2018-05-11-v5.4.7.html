New Features:

  • Completely reworked the way cmdchannels work - it now functions more like the joinmsg and leavemsg commands (see commands page for full details)
  • Created new autorole command that allows roles to be automatically added to users immediately, after a set amount of time, or after they've sent a certain number of messages (see commands page for full details)
  • DBL post will now display the bot's server count

Bug Fixes and Adjustments:

  • Changed the formatting of the Connect Four display to include column numbers
  • Fixed a Connect Four bug preventing users from filling columns fully
  • Commands will now no longer execute while the bot is booting, as intended
  • Fixed a bug that caused reminders to trigger several times instead of once
  • Resolved an issue with birthdays that prevented the bot from sending Happy Birthday messages to guilds
  • birthdays can now only be accepted in the format "-number- -written month-" (e.g. "22 Feb"), as they should be, and the error message is now more comprehensive
  • Fixed a problem that made birthdays trigger even if it isn't actually the user's birthday
  • Connect Four will now only run if the bot has the permissions it needs
  • remindme messages can now be up to 2000 characters (used to be 100)
  • Restricted ship command to users so that it wouldn't break anymore
  • Channel based configuration options have been remade in a better way
  • Altered paginated list formats
  • Sprinkled with love and affection

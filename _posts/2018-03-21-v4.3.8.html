New Features:

  • daily command now gives bonus tokens based on the streak duration, +5 for each successive day

Bug Fixes and Adjustments:

  • Slightly altered the way info and **help embeds are designed
  • Fixed a bug in which the bot was a bit too enthusiastic about reminding people and would send reminders after every reboot
  • remindme command is now accurate to the nearest second instead of the nearest minute, as it should be
  • The reminder command should also now correctly display the time set (e.g. 2 hours and 5 minutes instead of 125 minutes and 0 seconds)
  • Time set for the reminder command is now significantly more forgiving (ignores minor spelling mistakes, accepts non-chronological combinations eg. 5h3d2m, etc)
  • Removed boot, card, and stomach from insult nouns because they didn't make sense.
  • Added fart to insult nouns because I have the humour of a 3rd grader
  • Made the daily command prettier
  • Fixed a bug that caused the bot to shut down a lot more frequently than it needed to.

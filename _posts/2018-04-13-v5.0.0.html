New Features:

  • Added guildinfo command which retrieves guild stats

Bug Fixes and Adjustments:

  • Automatic Log/Msg channel setting is no longer logged unnecessarily frequently
  • Renamed commandchannel command to cmdchannel
  • fortnite profile now returns "No games played." on pages where the user hasn't played any games instead of entirely excluding that page (Also had to disable all time-based stats because of Fortnite's servers)
  • cah join confirmation message sent via DMs is now the right colour and format
  • The user stat returned by info was slightly inaccurate and has been updated to address this issue
  • Moved userinfo to the Information module
  • Changed quote timestamp so that its day/month/year instead of being regional
  • Grouped all of the joingmsg and leavemsg commands
  • fortnite commands will now also inform the user when the servers are down instead of breaking when trying to access stats
  • Changed the perms Cinnamon requests to reduce paranoia about malicious intent
  • Attempting to execute a command which requires a permission that Cinnamon doesn't have will return an error now
  • Log channels and Message channels which the bot no longer has access too will now automatically be overwritten
  • Fixed a problem with update message broadcasting
  • Sprinkled with love and affection

New Features:

  • Completely reworked the bot's foundation from scratch - you guys won't notice anything, but it took a heck of a lot of work xD
  • THE HELP AND CHANGELOG PAGES NOW PLAY MUSIC AND LOOK COOLER! W O A H

Bug Fixes and Adjustments:

  • userinfo now puts a crown on the profile of the server owner
  • Fixed a bug where getting the profile of a user with no roles would give an error
  • The "Permissions" section of userinfo now works properly (shows the names of the permissions the user has instead of a number)
  • Corrected an issue that caused the embed color of the userinfo command to be incorrect (should always be the same colour as the user being displayed)
  • quote will now delete the message that invoked it (e.g. if you type ~quote 2132142412131 it'll delete your message then quote the message with the ID that was provided)
  • Join and Leave messages now mention the involved user by default, at the beginning of the message
  • Maximum message length increased from 50 to 350, excluding {user}
  • joinmsg list and leavemsg list are now paginated (multiple pages which you can use reactions to move through), with 5 messages per page
  • Fixed a bug where joinmsg list and leavemsg list showed empty lists
  • Solved a problem where users could add the same leave message multiple times
  • Corrected an issue in which leavemsg add was causing all kinds of problems if it was the first leave message to be added for a guild
  • grouphug and battle have been re-enabled
  • battle messages will now be deleted after the battle is over to prevent the message glitching out and lagging the entire channel
  • Sprinkled with love and affection

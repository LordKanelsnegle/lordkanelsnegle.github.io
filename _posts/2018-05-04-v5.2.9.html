New Features:

  • Connect Four (Game Update)

    I created an emoji-based, embedded Connect Four game which can be played using reactions as an adaptation of the well-known board game. The general concept is to get four tokens in a line, whether horizontally, vertically, or diagonally. Each user starts with a token of a particular colour (indicated by the emoji next to your username) and users take turns dropping one token at a time into the column of their choice.
    For more information and command usage see the commands page.

Bug Fixes and Adjustments:

  • Fixed a bug that caused that hourly and daily message activity not to be recorded
  • Removed the logchannel feature and commands because I realised that logging isnt a very unique feature and is already accomplished by many other bots, deeming it unnecessary - plus Audit Logs are a thing now
  • Fixed yet another database issue that caused the bot to crash when joining a new guild
  • Patched a Cards Against Humanity bug which prevented users from being able to use any game moderation commands like cancelling the game or ending it prematurely

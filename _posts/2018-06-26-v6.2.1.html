New Features:

  • Brand new website!! The two old websites have been migrated to and merged with this one, in hopes that this will enhance user experience. The home page still needs a bit of work and the commands page needs to be styled properly but for the most part everything looks good
  • Added a secondary help command which can be used to search up a link for a particular command instead of linking to the Commands page in general

Bug Fixes and Adjustments:

  • Fixed a problem with timed autoroles and reminders that caused the bot to crash for durations longer than 24 days, largely increasing performance in the process
  • Removed a chunk of autorole code that turned out to be completely useless (must’ve been written at 2am)
  • Split the message handler in half so that autoroles and guildstats are handled separately, hopefully this should reduce the amount of gateway blocking that occurs
  • Enhanced the way connect4 handles reactions being added, increasing performance significantly
  • Reworked the battle command from the ground up - it now behaves like the rest of the games, with similar commands and display formats as well as improved efficiency/reliability/readability
  • Cleaned up other game formats so that they all have a consistent format
  • Bot will no longer send annoying “Commands cannot be used in this channel” on the support server (wont affect most people)
  • Statuses involving users will no longer include me as I’d like them to reflect the community :3
  • The ship command now ships based on nickname instead of username
  • Shipnames which are the same as one of the input names are now excluded
  • The meters for the ship, gaydar, and waifurate commands now display based on rounded values instead of always rounding down
  • Cinnamon is now un-roastable :')
  • Fixed a bug that allowed the birthday command be abused for grinding tokens
  • Resolved an issue which wouldn't display the player list for connect4 lobbies
  • Added cf as an alias for connectfour
  • The connect4 gameover message now displays "The game was ended prematurely." instead of saying one of the users won, if the game was ended prematurely
  • Addressed a problem with connect4 that prevented the reaction for the 7th column from showing up
  • You now lose 3 MMR for each player lost to in a battle instead of 5
  • Expanded upon the battle info command to include an explanation of how the MMR awarding and ranking systems work
  • All game commands now have ' ' as an alias for info (meaning that typing only the name of the game will display the information)
  • Edited the update broadcast so that it links to the new changelog page
  • Fixed a cah bug so that it no longer sends a "players left" embed when there are no players left that need to play their turn
  • The embed sent to cah players in DMs with their hand of cards now prompts users to use "cah pick" in the game channel to make it more noob-friendly
  • Links displayed by the help command have been updated to reflect the website changes described above
  • Resolved an issue where quote could be used on messages which contain only an embed (and nothing actually quotable)
  • The error msg displayed when too many or too few parameters are used is now more informative and prompts the user to use the command-specific help command
  • Added dl as an alias for dicklength
  • Fixed a bug where the reminder command would say that the timespan entered is invalid when in reality there was just no message being set as the reminder
  • The avatar command should no longer break when used on users with no avatar set
  • Sprinkled with love and affection
